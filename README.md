### 3. Rule

1. 매주 해당 챕터 공부해오기
2. 스터디날에 각자 맡은 Story발표
   1. 랜덤으로 준비해온 Story 발표
3. 스터디 날짜는 기본적으로 `[화요일 저녁 8시]`
4. 첫 스터디 진행일: 8/1(화)
5. 발표 시 간략한 발표자료 필수(요약본 정도는 레포에 미리 올릴 것)
6. 모든 공부 자료 및 발표 자료는 본인 디렉토리에 저장

- 너무 깊이 공부하지 않는게 포인트
